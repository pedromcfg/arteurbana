# Pasta de Dados

Esta pasta contém os dados dos artistas que são carregados automaticamente pelo site.

## Ficheiro: `artists.json`

Este ficheiro contém todos os dados dos artistas, incluindo as imagens em formato base64.

### Como atualizar:

1. Aceda ao backoffice (`backoffice.html`)
2. Adicione ou edite os artistas
3. Clique em "Exportar para GitHub"
4. Substitua o ficheiro `data/artists.json` com o ficheiro descarregado
5. Faça commit e push para o GitHub

O site carregará automaticamente os dados deste ficheiro quando alguém acessar o site.
